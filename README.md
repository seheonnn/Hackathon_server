# 그린퀘스트 ‘Green Quest’ 

# ✏️ 프로젝트 요약
- 혼자 하는 여행도 즐겁게! 여행지에서 보물 찾기!
- 국내 여행 중 사용자의 위치를 기반으로 쪽지 미션을 수행할 수 있습니다.
- 쪽지에 적힌 미션을 수행하여 사용자의 식물을 키우는 보상형 여행 서비스입니다.
  
# 🎯 프로젝트 목표
- DTO Inner Class 사용
- [Amazon AWS, RDS](https://aws.amazon.com/ko/)를 이용한 ubuntu 환경 배포
- [4th UMC 해커톤] T팀 백엔드 최우수상✨
